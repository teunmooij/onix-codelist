ONIX Code Lists Issue 58, July 2022

List 18: Person / organization name type

Value Description Notes
00 Unspecified Usually the name as it is presented on the book
01 Pseudonym May be used to give a well-known pseudonym, where the primary name is a ‘real’ name
02 Authority-controlled name
03 Earlier name Use only within <AlternativeName>
04 ‘Real’ name May be used to identify a well-known real name, where the primary name is a pseudonym
05 Transliterated / translated form of primary name Use only within <AlternativeName>, when the primary name type is unspecified, for names in a different script or language
06 Later name Use only within <AlternativeName>
07 Fictional character name Use only within <NameAsSubject>, to indicate the subject is fictional. For use in ONIX 3.0 only
