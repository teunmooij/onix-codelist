ONIX Code Lists Issue 58, July 2022

List 19: Unnamed person(s)

Value Description Notes
01 Unknown
02 Anonymous Note that Anonymous can be interpreted as singular or plural
03 et al And others: additional contributors not listed
04 Various Use for example when the product is a pack of books by different authors
05 Synthesised voice – male Use with Contributor role code E07 ‘read by’, eg for audio books
06 Synthesised voice – female Use with Contributor role code E07 ‘read by’, eg for audio books
07 Synthesised voice – unspecified Use with Contributor role code E07 ‘read by’, eg for audio books
08 Synthesised voice – based on real voice actor Use with Contributor role code E07 ‘read by’, eg for audio books, and provide name of voice actor in <AlternativeName>. For use in ONIX 3.0 only
