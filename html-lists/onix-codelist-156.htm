ONIX Code Lists Issue 58, July 2022

List 156: Cited content type

Value Description Notes
01 Review The full text of a review in a third-party publication in any medium
02 Bestseller list
03 Media mention Other than a review
04 ‘One locality, one book’ program Inclusion in a program such as ‘Chicago Reads’, ‘Seattle Reads’, ‘Canada Reads’, ‘One Dublin, one book’
05 Curated list For example a ‘best books of the year’ or ‘25 books you should have read’ list, without regard to their bestseller status
06 Commentary / discussion For example a third party podcast episode, social media message, newsletter issue, other commentary (see also code 03 for very brief items)
